# Prep4Exam

AI-powered study assistant that turns raw notes into clear explanations, quizzes, and personalized feedback.

## 🚀 Features

- 📖 Convert notes into simple explanations
- 🧠 Generate AI-based quizzes
- ✅ Review answers with detailed feedback
- 🎯 Identify weak areas and revision strategy

## 🛠 Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Google Gemini API

## ⚙️ Setup

1. Clone the repo:
   ```bash
   git clone https://github.com/Saurabhvch12003/prep4exam.git
   cd prep4exam

2. Install dependencies:

    npm install

3. Add environment variables:

   GEMINI_API_KEY=your_api_key_here

4. Run the app:

   npm run dev
